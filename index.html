Tugas Praktikum: Konfigurasi Dasar MikroTik

Dipublikasikan oleh: Guru Produktif TKJ | Kelas: XI TKJ 2

Silakan baca modul berikut sebelum melakukan praktikum di lab komputer. Pastikan kabel UTP (Straight) sudah disiapkan.

Langkah 1: Akses Winbox

Colokkan kabel dari PC ke Ether 2 pada router MikroTik. Buka aplikasi Winbox, pilih tab Neighbors, dan klik MAC Address router. Login dengan user admin dan password kosong.

Langkah 2: Setting IP Address

Masuk ke menu IP > Addresses. Tambahkan IP baru untuk Ether 2, misalnya 192.168.10.1/24. Klik Apply dan OK.

Penting: Jangan lupa kerjakan kuis di bawah setelah selesai membaca.
